
# wnckprop / example-list


## list all windows

``` sh
$ wnckprop --list
```

## list windows by workspace

run

``` sh
$ wnckprop --list --workspace=0
```

or

``` sh
$ wnckprop --list --workspace=1
```

or

``` sh
$ wnckprop --list --workspace=2
```

or

``` sh
$ wnckprop --list --workspace=3
```

show like

```
12582917: pcmanfm-qt
20971526: LXQt Panel
```
